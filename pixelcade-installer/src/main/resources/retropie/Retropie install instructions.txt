Pixelcade Install Instructions RetroPie:

Refer to setup instructions at step 11 https://www.instructables.com/id/Vertical-Bartop-Arcade-With-Integrated-PIXEL-LED-D/

Summary of steps:

1. Install RetroPie on your Pi

2. Install Java 8 on your Pi

3. Copy retrogame.cfg to your Pi /boot/retrogame.cfg

4. Copy runcommand-onstart.sh and runcommand-onend.sh to your Pi /opt/retropie/configs/all/
